# 账单的相关操作

建议统一支付出口，比如只用**微信**，或者只用**支付宝**，这样在汇总账单的时候比较方便，不用到处找。  
再有就是最好每周记录一下本周的支付记录。以免积压太多都不想动了。

虽然支付宝和微信都有自己的账单统计管理，但我还是觉得这个比较清楚方便。


## 1. 新建账单

日记支持账单系统，但需要遵循严格的输入格式。

1. 新建日记
2. 选择类别为 `账单`
3. 此时日记标题会自动填写上 `账单` 二字，当然这个标题可以为任意，你可以记录挑选此次账单中比较重要的条目作为日记标题。
4. 日记的内容就是账单的内容，它的书写格式为 `条目+空格+花费金额`
    ```
       早餐 -5
       午餐 -12
       晚餐 -20
       超市-圣豪 -20
       摩托车加油CBR650R#92 -75.4
       闲鱼卖iPhone +3500
    ```
    __注意__：只要金额不是以 `-` 开头的，都是收入，没有 `+`号也算收入，比如下面内容等价
    ```
    闲鱼卖iPhone +3500
   
    闲鱼卖iPhone 3500
    ```

## 2. 账单条目缓存
账单记起来会有点繁琐，有时候一些每天都会有的条目就不想每天都打全了。  
为了更方便的记录账单，当当前编辑的日记类别是账单时，编辑区会自动匹配以前存在过的账单条目。  
列出的条目可用鼠标选择，也可以直接用条目前方的序号选择

![账单提示](./img/bill_prompt.png)

正常情况下，每次登录都会将原有账单条目缓存至本地，如果想手动获取，可以进入账单页面点击按钮 **获取最新账单条目**，在下一张截图中能看到这个按钮。

## 3. 账单查看

账单记录完成之后，可以通过查看账单页面来概览第一个月的账单概览。

![账单概览](./img/bill_brief_month.png)

点击列表中的日期可直接进入对应日记内容
